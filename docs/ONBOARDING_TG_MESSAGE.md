# Сообщение для Telegram — установка с нуля

> Отправлять по одному сообщению, каждый блок = отдельное сообщение в ТГ

---

## Сообщение 1

```
Шаг 1 из 4 — Открой Терминал

На маке нажми Cmd + Пробел, напиши Terminal, нажми Enter.
Откроется чёрное окно — это терминал. Сюда будешь вставлять команды.
```

## Сообщение 2

```
Шаг 2 из 4 — Установи инструменты разработчика

Скопируй эту строку, вставь в терминал (Cmd+V), нажми Enter:

xcode-select --install

Появится окно — нажми "Install". Подожди пока закончится (5-10 мин).
```

## Сообщение 3

```
Шаг 3 из 4 — Установи Homebrew и Node.js

Скопируй и вставь в терминал:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Может попросить пароль — это пароль от мака (от компьютера). При вводе символы не видны — это нормально, просто набери и нажми Enter.

Когда закончит — вставь следующее:

brew install node

Потом:

npm install -g @anthropic-ai/claude-code
```

## Сообщение 4

```
Шаг 4 из 4 — Запусти Claude Code

Напиши в терминале:

claude

Он попросит залогиниться — зайди через аккаунт claude.ai

После входа напиши ему: setup

Дальше он сам всё доустановит. Потом можно писать ему задачи на русском.
```
