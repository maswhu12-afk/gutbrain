# GutBrain — Начни здесь

> 4 шага — и ты готова работать над приложением

---

## Шаг 0: Зарегистрируйся на GitHub (5 минут)

GitHub — это место где хранится код проекта. Нужен аккаунт.

### 0.1 Создай аккаунт

1. Открой https://github.com/signup
2. Введи email, придумай пароль, выбери username
3. Подтверди email (придёт письмо)

### 0.2 Установи Git (если ещё нет)

Открой Терминал (`Cmd + Пробел` → `Terminal` → Enter) и выполни:

```bash
git --version
```

Если показало версию — уже установлен. Если нет:

```bash
xcode-select --install
```

### 0.3 Настрой Git

Замени имя и email на свои:

```bash
git config --global user.name "Твоё Имя"
git config --global user.email "твой@email.com"
```

## Шаг 1: Установи Claude Code (5 минут)

Claude Code — это AI-помощник который живёт в терминале. Он читает код, пишет код, запускает команды. Ты пишешь ему что нужно — он делает.

### 1.1 Открой Терминал

- На Mac: нажми `Cmd + Пробел`, напиши `Terminal`, нажми Enter
- Откроется чёрное окно с мигающим курсором — это терминал

### 1.2 Установи Node.js (если ещё нет)

Скопируй и вставь эту команду в терминал, нажми Enter:

```bash
brew install node
```

Если brew не найден, сначала установи его:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 1.3 Установи Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

### 1.4 Проверь

```bash
claude --version
```

Должно показать версию. Если показало — Claude Code установлен.

## Шаг 3: Скажи "setup" (10-15 минут)

После запуска Claude Code просто напиши:

```
setup
```

Claude Code **сам** установит всё что нужно:
- Xcode Command Line Tools
- Homebrew (менеджер пакетов)
- Node.js (для мобильного приложения)
- Python (для сервера)
- Expo (фреймворк для iPhone приложения)
- Все зависимости проекта
- Шаблон конфигурации

Сиди и смотри. Когда закончит — покажет чек-лист что установлено.

## После установки

### Установи Expo Go на iPhone

1. Открой App Store на iPhone
2. Найди "Expo Go"
3. Установи (бесплатно)
4. Это позволит запускать приложение прямо на телефоне

### Запусти приложение

В Claude Code напиши:

```
запусти приложение
```

Или вручную:

```bash
cd app && npx expo start
```

Появится QR-код → отсканируй камерой iPhone → приложение откроется в Expo Go.

### Что можно попросить Claude Code

```
Покажи что есть в проекте
Прочитай продуктовый спек и расскажи коротко
Создай экран онбординга
Добавь AI чат на главный экран
Сделай экран meal plan
```

## Если что-то пошло не так

- **"command not found: brew"** — установи Homebrew (команда выше в шаге 1.2)
- **"command not found: node"** — установи Node.js: `brew install node`
- **"command not found: git"** — установи git: `brew install git`
- **Просит пароль** — это пароль от Mac (от компьютера), не от Claude
- **Что-то непонятное** — скриншот → Анатолию
